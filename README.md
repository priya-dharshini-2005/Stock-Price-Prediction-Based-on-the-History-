# Stock Price Forecasting Using GRU

## Project Overview

This project predicts future stock prices using a Gated Recurrent Unit (GRU) neural network based on historical stock market data.

## Features

- Historical stock price analysis
- Data preprocessing and normalization
- GRU-based deep learning model
- Future stock price prediction
- Performance evaluation

## Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

## Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Scaling
4. Sequence Creation
5. GRU Model Training
6. Stock Price Prediction
7. Performance Evaluation

## Model Architecture

- Input Layer
- GRU Layers
- Dense Output Layer

## Evaluation Metrics

- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)

## Future Enhancements

- Hyperparameter Tuning
- Bidirectional GRU
